# Data Cleaning in Excel

## Project Overview
This project focuses on cleaning raw data using **Microsoft Excel** to make the dataset accurate, organized, and ready for analysis. The goal was to identify errors, handle missing values, standardize formats, and improve overall data quality.

## Objective
The objective of this project is to transform messy raw data into a clean and structured dataset that can be used for reporting, dashboard creation, and data analysis.

## Data Cleaning Steps Performed

### 1. Removed Duplicate Records
- Identified duplicate rows in the dataset.
- Used **Remove Duplicates** feature in Excel to delete repeated records.

### 2. Handled Missing Values
- Found blank or null values in important columns.
- Replaced missing values manually or using logical formulas.

### 3. Standardized Data Format
- Converted inconsistent date formats into a single format.
- Standardized text values (Example: "Male", "male", "MALE" → "Male").

### 4. Fixed Incorrect Data Types
- Converted numbers stored as text into proper numeric values.
- Corrected currency and percentage formats.

### 5. Cleaned Text Data
- Removed extra spaces using `TRIM()`.
- Fixed capitalization using `PROPER()`, `UPPER()`, and `LOWER()`.

### 6. Handled Special Characters
- Removed unwanted symbols or unnecessary characters from data fields.

### 7. Created New Calculated Columns
- Generated new columns using formulas for better analysis.
- Example:
  - Total Sales = Price × Quantity
  - Profit Margin = Profit / Revenue

### 8. Used Power Query for Transformation
- Imported raw dataset into Power Query Editor.
- Applied transformations such as:
  - Splitting columns
  - Replacing values
  - Changing data types
  - Filling missing values

### 9. Data Validation Check
- Verified cleaned data for consistency and accuracy.
- Ensured no duplicate, blank, or invalid entries remained.

## Final Outcome
After cleaning, the dataset became:
- Accurate  
- Consistent  
- Ready for analysis  
- Suitable for dashboard creation and business insights
